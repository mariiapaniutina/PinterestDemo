# PinterestDemo
This is the demo for fashion discovery platformwith design based on http://pinterest.com

Was used: 
* jquery.js 
* mustache.js
* bootstrap

JSON contains random data, generated at https://www.mockaroo.com/
```ruby
JSON structure =>
{
   "title": "<TITLE>",
   "blurb": "<BLURB>",
   "author": "<AUTHOR>",
   "thumbnail_url": "<URL>",
   "details_url": "<URL>"
}
```

###Installation
Download, unzip, run **index.html**
